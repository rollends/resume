\pdfbookmark[1]{Projects}{projects}
\ul{\Large\textbf{projects}}
\vskip 0.25em
\emph{Rollen likes to write code. Here is an excerpt of interesting projects Rollen has worked on over the years. All projects listed here can either be found in his Github profile, or the source code can be requested.}
\vskip 0.75em
\begin{adjustwidth}{1em}{0em}%
  \textbf{BurritOS: A Real Time Operating System} [\href{https://github.com/rollends/burritOSPrivate}{Github}]\hfill
  \begin{adjustwidth}{1em}{0em}
    Implemented (hard) real time operating system with message passing in C, from the ground up, for the purpose of autonomously managing trains on a physical track.
  \end{adjustwidth}
  \vskip 0.75em

  \textbf{Red Prism: Reddit Graph Analyzer} [\href{https://github.com/rollends/red-prism}{Github}]\hfill
  \begin{adjustwidth}{1em}{0em}
    Python project that downloads Reddit public post \& user information and analyzes the graph relationships present in a community to identify ``popular'' or ``interesting'' content.
  \end{adjustwidth}
  \vskip 0.75em

  \textbf{Lua on Serious Threads (LoST)} [\href{https://github.com/rollends/lost}{Github}]\hfill
  \begin{adjustwidth}{1em}{0em}
    Allows for many Lua tasks to be performed in parallel on a number of kernel-level worker threads.
    Task execution is interleaved by modifying the Lua 5.3.4 runtime to support the interruption of a Lua task at pre-defined intervals.
  \end{adjustwidth}
  \vskip 0.75em
  \textbf{Implementation of Transverse Feedback Linearizing Controllers} [\href{https://github.com/rollends/SE499}{Github}]\hfill
  \begin{adjustwidth}{1em}{0em}
    Undergraduate research project investigating two different implementations of path following controllers and their practicality in terms of performance and numerical precision.
  \end{adjustwidth}
\end{adjustwidth}
